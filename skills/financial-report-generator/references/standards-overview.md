## Key differences by standard

### Balance Sheet structure

| Standard | Assets order | Liabilities order |
|---|---|---|
| IFRS | Non-current first, then current | Non-current first, then current |
| US GAAP | Current first, then non-current | Current first, then non-current |
| OHADA | Fixed assets first, then current | Permanent capital first, then debt |

### Mandatory statements

| Standard | Required statements |
|---|---|
| IFRS | Statement of Financial Position · P&L · OCI · Changes in Equity · Cash Flow · Notes |
| US GAAP | Balance Sheet · Income Statement · Changes in Equity · Cash Flow · Notes |
| OHADA | Bilan · Compte de résultat · Tableau de flux de trésorerie · Notes (ETIC) |

### Revenue recognition

- **IFRS 15**: performance obligations, 5-step model
- **ASC 606** (GAAP): equivalent to IFRS 15, US-specific guidance
- **OHADA**: accrual basis, specific rules for long-term contracts

## Materiality thresholds

Apply judgment — these are common starting points, not hard rules:

| Benchmark | Typical threshold |
|---|---|
| Total assets | 1-2% |
| Revenue | 1-2% |
| Net income | 5% |
| Equity | 5% |

Items above the threshold require separate line item disclosure.
Items below may be aggregated.
